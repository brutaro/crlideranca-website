## 🚀 Como testar

### Opção 1: Script automático
```bash
./start-server.sh
```

### Opção 2: Servidor manual
```bash
# Python 3
python3 -m http.server 8000

# Node.js
npx serve . -p 8000

# PHP
php -S localhost:8000
```

### URLs de teste
- **Site completo**: http://localhost:8000/index.html
- **Versão teste**: http://localhost:8000/test-site.html
